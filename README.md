# RoadOfShader

Unity版本：Unity 2020.1.4f1

这里主要记录自己在学习图形学的过程中的一些练手项目，都是通过Unity的新渲染管线URP来实现的。

其中，很多项目是参考网上的博客之类的资源，使用URP实现原本的Buildin管线的项目，相应的参考链接会一并列出，同时也感谢这些博主的分享。

限于篇幅，这里只会给出一个链接，详细的文档会按照一定的章节，写成一篇完整的博客。
